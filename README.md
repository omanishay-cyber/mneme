# Mneme

Local persistent memory + code graph for AI coding tools.
49 tools · 11 scanners · 14 WebGL views · 100% local.

## Docs & install

The home for Mneme is **[mneme.stryxlabs.com](https://mneme.stryxlabs.com)**.

### Install

**macOS / Linux**
```
curl -fsSL https://stryxlabs.com/install.sh | bash
```

**Windows (PowerShell)**
```
iex (iwr https://stryxlabs.com/install.ps1).Content
```

That's it — both scripts auto-detect platform/arch, verify SHA-256, and install to `~/.mneme`.
